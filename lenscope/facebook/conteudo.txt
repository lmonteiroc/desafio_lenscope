Sobre os conteúdos visuais

A   Landing Page foi desenvolvida no Visual Studio code, como extensão para escrita utilizou-se Emmit, Drácula e auto complete HTML CSS e JS, 
os Icons foram retirados do site https://icon-icons.com/, para adaptação da página aos diversos tamanhos de tela foi utilizada a mídia @media (max-width )
com ajuste entre 820 até 950px, as linguagens utilizadas na página são: HTML CSS e JS.

Para edição da imagem foi utilizado o paint 3D foi aplicada cor preta no fundo, um filtro frio cor lavanda, a fonte gotic bald e gotic bald italic para a
escrita na chamada da imagem, a figura foi retirada do google imagens.
